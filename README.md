# Aurethium

Portfolio/demo version of a luxury crypto investment platform 
built for a private client.

> Note: This is a sanitised demo version. 
> The production deployment is privately hosted.

## Live Demo
[link to old version or deploy it fresh on Cloudflare Pages]

## Tech Stack
- Frontend: HTML5, CSS3, Vanilla JS
- Backend: Node.js, Express
- Database: Supabase (PostgreSQL)
- AI Assistant: Anthropic Claude API
- Deployment: Render (API), Cloudflare Pages (frontend)

## Key Features
- AI-powered investment assistant (Claude API)
- Waitlist and contact backend
- REST API with Express
- PostgreSQL via Supabase
