Aurethium
Pre-launch crypto investment platform template with AI chatbot integration.

A full-stack crypto investment platform built as a purchasable template. Features a complete frontend UI, Node.js backend, Supabase database, and an integrated Claude AI chatbot for user assistance. Available as a white-label solution for crypto investment businesses.

Note: This is a template product. The investment transaction flow is not live. Businesses purchasing this template are responsible for integrating their own payment and compliance infrastructure before going live with real user funds.

Stack
Frontend: HTML, CSS, Vanilla JavaScript
Backend: Node.js / Express
Database: Supabase (PostgreSQL)
AI Chatbot: Claude API (Anthropic)
Deployment: Render (backend) / Static hosting (frontend)
Features
User registration and authentication
Investment dashboard UI
Portfolio overview and holdings display
AI-powered chatbot assistant (Claude API)
Admin panel
Crypto price display integration
Fully designed UI ready for white-label customization
